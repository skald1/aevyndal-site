# ÆVYNDAL SEARCH SYSTEM

## Purpose

Search is king.

Search is not a utility.

Search is discovery.

People should be one interaction away from something interesting.

---

# Philosophy

Search should answer:

"What else belongs here?"

before:

"What exactly matches?"

Curiosity matters more than precision.

---

# Search Groups

Locations

People

Creatures

Faiths

Articles

Campaigns

Groups are always visible.

---

# Search Overlay

Soft.

Glass.

Never full-screen black.

World remains visible.

Search feels inviting.

Not urgent.

---

# Result Cards

Show:

Category

Title

Excerpt

Related Content

Results should create journeys.

Not dead ends.

---

# Empty States

Avoid:

"No results."

Prefer:

"Nothing here yet."

"Try exploring nearby places."

---

# Search Technology

Pagefind

Entirely static

Fast

Accessible

No external dependency

---

# Search Is Always Available

Floating nav

Ghost header

Dedicated page

Keyboard shortcut

Search should never be buried.

---

# Final Rule

Search should create discovery.

Not merely retrieval.
