# ÆVYNDAL MEDIA SYSTEM

## Purpose

Media exists to support atmosphere.

Media should create emotion.

Media should never compete with information.

The world is larger than the archive.

The archive floats inside the world.

---

# Philosophy

Media is memory.

Not spectacle.

Not decoration.

Not content.

The best media is felt before it is noticed.

---

# Hierarchy

World

↓

Atmosphere

↓

Glass

↓

Information

Media belongs primarily to the world layer.

---

# Images

Images should feel remembered.

Avoid:

* splash art
* character posters
* obvious framing
* isolated portraits

Prefer:

* landscapes
* environments
* weather
* architecture
* ritual
* work
* travel
* communities

The question every image answers:

"Who could I be here?"

---

# Ambient Video

Ambient video replaces static imagery wherever possible.

Video is motion texture.

Not content.

Not narrative.

---

# Good Video Subjects

Fog moving through trees

Snowfall

Rain on glass

Stars

Traffic reflections

Candle flames

Dust motes

Pollen

Leaves

Cloud shadows

Water ripples

Smoke

Wind through grass

---

# Avoid

Combat

Dialogue

Action

Fast movement

Camera shake

Explosions

Jump cuts

Busy crowds

Faces

Talking

Trailers

Spectacle

---

# Technical Requirements

Muted

Autoplay

Loop

Playsinline

No controls

Poster image fallback

Reduced motion support

---

# Duration

5–12 seconds

Loop should be invisible.

No obvious restart point.

---

# Processing

Desaturate

Darken

Reduce contrast slightly

Subtle grain

Slow speed if necessary

Avoid over-sharpening

---

# Formats

Preferred:

webm

Fallback:

mp4

Poster:

webp

---

# Storage

Cloudflare R2

Future:

Cloudflare Stream

Media paths should remain abstracted.

Never hardcode.

---

# Hero Media

Full width.

Large negative space.

Text-safe center.

Darkened.

Never distract from typography.

Should feel like standing somewhere.

Not watching something.

---

# Card Media

Image and content should blend.

No hard line.

Media fades into card body.

Cards are windows.

Not boxes.

---

# Relationship Between Pages

Media should inherit mood.

Example:

Mütvia page:

Related cards should contain:

* branches
* fog
* moonlight

Kill City page:

Related cards should contain:

* neon
* rain
* windows

Everything belongs to the same scene.

# Motion

Motion should be slow.

Everything drifts.

Nothing snaps.

Nothing demands attention.

Motion should be subconscious.

---

# Image Philosophy

Images are memories.

Not illustrations.

Not wallpapers.

Not posters.

People should feel:

"I've been there."

Not:

"That's pretty."

---

# Accessibility

Respect reduced motion.

Fallback to posters.

Site must remain beautiful without video.

Video is enhancement.

Not dependency.

---

# Final Rule

Media should create atmosphere.

Atmosphere should create belonging.

Belonging is the goal.
