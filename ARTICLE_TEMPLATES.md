# Purpose

This document defines how writers create content.

Articles should answer questions.

Not display information.

Present before history.

Identity before exposition.

---

# Universal Order

1. What is it?
2. What is it like?
3. Why should I care?
4. Who lives here?
5. What threatens it?
6. What stories happen here?
7. History

History always comes last.
